Apa Itu Algoritma?

Algoritma merupakan prosedur yang mengikuti serangkaian langkah yang sistematis yang mengarah pada jawaban yang diinginkan

  • SEARCHING

Apa itu Searching?

Searching adalah suatu kegiatan memindai atau mencari suatu data (bisa satu atau lebih) dari sekumpulan data atau dari bayaknya data yang ada

Learn More

Linear Search

Linear search (pencarian berurutan) adalah suatu Teknik pencarian data dalam array 1 dimensi yang menelusuri semua elemen-elemen array dari awal sampai akhir dan pencarian akan berhenti ketika data ditemukan.Tetapi jika sampai akhir elemen data tidak ditemukan maka pencarian akan tetap selesai dan hasilnya adalah 0.

Binary Search

Binary Search (metode pencarian bagi dua) adalah algoritma pencarian data dari array terus menerus sampai datanya ditemukan. Data yang akan dicari menggunakan binary search harus terurut secara ASCENDING atau DESCENDING

Sorting

Sorting adalah proses pengaturan elemen elemen dalam suatu himpunan sehingga di susun berdasarkan urutan tertentu.

Babble Sort

Bubble sort adalah salah satu algoritma sorting sederhana yang membandingkan secara berpasangan Setiap elemen elemen dalam suatu himpunan data dan menukar posisi mereka jika mereka tidak dalam urutan yang benar

Insertion Sort

Algoritma sederhana untuk pengurutan sekumpulan data dengan membandingkan semua data mulai dari data yang ke 2 dengan data ke 1 dan mulai dari data nya terus bertambah sampai bberapa jumla sekumpulan data dan di bandingkan data nya sampai data ke 1

Big O, Big Omega, Big Tetha

Fungsi t(n) dikatakan berada di dalam O(g(n)), dinyatakan dengan (n) O(g(n)), apabila t(n) dibatasi atas oleh beberapa konstanta dikali g(n) untuk seluruh n, jika ada beberapa konstanta c positif dan beberapa bilangan bulat tak negatif N0 sedemikian sehingga: (n) ≤ cg(n) untuk semua n ≥ n0

Big O

Definisi 2. T(n) = Ω(g(n)) dibaca "T(n) adalah Omega (f(n)" yang artinya T(n) berorde paling kecil g(n)) bila terdapat konstanta C dan n0 sedemikian sehingga T(n) ≥ C(f (n)) untuk n ≥ n0.

Big Omega

Definisi 3. T(n) = Θ(h(n)) (dibaca "T(n) adalah tetha h(n)") yang artinya T(n) berorde sama dengan h(n) jika T(n) = O(h(n)) dan T(n) = Ω(h(n)). • Jika T(n) = Θ(h(n) maka kita katakan T(n) berorde h(n)
